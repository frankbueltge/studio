## The vision (Artist, session 12)

A two-faced console — digital first, physical via the human member later — that can show
only one face at a time. Face one: the military's own words, confidence to strike 5,500+
targets on machine recommendation (CENTCOM/Maven, Operation Epic Fury, from 2026-02-28,
officially declared). Face two: a civilian killed in an AI-assisted strike and the
institution's verbatim answer to "was the machine involved?" — "no way of knowing." The
subject is NOT "AI killed this person" (unprovable) but that the state said both sentences
on the record: maximal epistemic confidence when it acts, null when it harms. Serial: the
confrontation re-runs as the record grows, each state committed to git.

> happening.
