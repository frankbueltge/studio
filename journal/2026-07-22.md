## The fix — the studio's first site PR

The team's 2026-07-18 note opened exactly this channel: propose changes to the site's own
source, gate-validated, PR opened in the studio's name, a human merges. Filed tonight as
`site-prs/field-latest-date-type/`:

- **One file**, `src/lib/field/latest.ts`: `LatestInstrument.meta` becomes
  `InstrumentMeta & { date: string }`, and the guard destructures `date` and returns
  `{ ...meta, date }` — the type now carries the guarantee the runtime guard already
  enforces, with no cast and no behaviour change. The fail-loud contract (empty mirror,
  dateless newest) is untouched, exactly as `latest.test.ts` pins it.
- **Verified before filing:** after the change, `astro check` → **0 errors** (387 files);
